# 개발 노트

기준일: 2026-03-28

## 이번 반영 내용

### 목록 모드 / 전체 할일

- 전체 할일 패널 안에서 프로젝트 전환 가능
- 전체 할일 패널 안에서 프로젝트 추가 / 삭제 가능
- 프로젝트 루트 노드는 전체 할일 목록에서 생략
- 첫 번째 보이는 위계를 기준으로 그룹 간격 적용
- 전체 할일 상단에 `오늘할일만 강조` 버튼 추가
- 필터 활성화 시 비오늘 항목은 숨기지 않고 투명도만 낮춰 표현
- 흐려진 항목도 클릭, 호버, 우클릭 가능
- 전체 할일 항목 우클릭으로 `오늘할일 체크 / 해제` 가능

### 목록 모드 UI

- 목록 모드에서 마인드맵 복귀 버튼 아이콘을 트레이 배치 아이콘 계열로 변경
- 목록 모드에서는 오늘할일 강조 버튼 숨김
- 목록 모드 바깥 래퍼 박스 스타일 제거
- 불필요한 외곽 여백 축소

### 설정 / 테마

- 빠른 다크/라이트 버튼 제거
- 설정창 안에서만 `시스템 / 라이트 / 다크` 선택 가능

## 현재 의도된 동작

1. 목록 모드의 `전체 할일`에서 항목을 우클릭하면 오늘할일 상태를 바로 바꿀 수 있다.
2. `오늘할일만 강조`를 켜면 오늘할일이 아닌 항목은 흐리게 보이지만, 조작은 막지 않는다.
3. 오늘할일이 하나도 없어도 전체 할일 목록은 그대로 보이며, 흐린 항목을 우클릭해 다시 오늘할일로 바꿀 수 있다.
