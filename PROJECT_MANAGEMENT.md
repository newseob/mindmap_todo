# 마인드맵 할일 관리 문서

기준일: 2026-03-28

## 1. 프로젝트 개요

`마인드맵 할일`은 정적 HTML 기반의 마인드맵형 할 일 관리 앱이다.  
프로젝트 단위로 마인드맵을 나눠 관리하고, 오늘할일, 메모, 목록 모드, 백업/복원, 테마 전환 기능을 함께 제공한다.

핵심 파일:

- `index.html`: 메인 앱 UI와 상태 로직
- `app-guide.html`: 서비스 소개 페이지
- `privacy.html`: 개인정보처리방침 페이지
- `ads.txt`: 광고 관련 설정 파일
- `mindmap_icon.svg`: 앱 아이콘

## 2. 현재 기능 정리

### 2-1. 마인드맵 편집

- 노드 추가, 삭제, 이름 변경
- 빈 공간 우클릭으로 독립 노드 추가
- 노드 연결 / 연결 해제
- 연결 모드 안내 배너 표시
- 단일 선택 및 `Shift` 다중 선택
- 드래그 박스 선택
- 다중 선택 노드 동시 이동
- 다중 선택 일괄 삭제
- 다중 선택 우클릭 메뉴 분기
- 오늘할일 노드 강조 보기

### 2-2. 프로젝트 관리

- 프로젝트별 독립 데이터 보관
- 프로젝트 추가 / 삭제 / 전환
- 우측 트레이에서 프로젝트 목록 관리
- 목록 모드의 `전체 할일` 패널 안에서도 프로젝트 전환 가능
- 목록 모드 프로젝트 패널에서 프로젝트 추가 / 삭제 가능

### 2-3. 오늘할일 / 메모

- 노드를 오늘할일로 지정 / 해제
- 오늘할일 패널 우측 체크 버튼으로 빠른 해제
- 오늘할일만 강조해서 보는 별도 토글
- 메모 패널 자동 높이 보정
- 메모 입력 폰트는 `Malgun Gothic` 사용

### 2-4. 목록 모드

- 마인드맵 모드와 별도인 목록 모드 제공
- 탭 구성:
  - 전체 할일
  - 오늘할일
  - 메모
- 목록 모드 상단에서 설정 열기 가능
- 목록 모드 상단에서 마인드맵 모드 복귀 가능
- 전체 할일은 트리 구조를 버튼 목록으로 표현
- 전체 할일에서 각 노드 옆 `+ / x` 호버 버튼으로 추가 / 삭제 가능
- 프로젝트 루트 노드는 전체 할일 목록에서 숨김 처리
- 첫 번째 보이는 위계 기준으로 그룹 여백 적용

### 2-5. 테마 / UI

- 라이트 / 다크 / 시스템 테마 지원
- 빠른 테마 토글 버튼은 제거됨
- 설정창 안에서만 테마 설정 변경
- 다크 모드에서 비선택 노드와 스크롤바 톤 별도 조정
- 트레이 숨김 / 표시
- 트레이 좌 / 우 이동
- 목록 모드일 때 외곽 래퍼 박스 스타일 제거

### 2-6. 백업 / 불러오기

- 현재 상태를 JSON 파일로 저장 가능
- JSON 파일 불러오기 가능
- 불러오기는 현재 데이터를 덮어쓰는 방식
- 불러오기 전 경고창 표시

### 2-7. 서비스 / 페이지

- 서비스 소개 페이지 분리
- 개인정보처리방침 페이지 분리
- 문의 이메일 연결
- 광고 영역은 현재 비활성화 상태

## 3. 최근 UI/기능 결정 메모

### 3-1. 목록 모드 방향

- 목록 모드는 전체 화면을 덮는 구조가 아니라 약 `500px` 폭의 오버레이 패널로 유지
- 탭은 `전체 할일`이 기본
- 외곽 래퍼 박스 느낌은 제거하고, 실제 패널들만 보이게 정리

### 3-2. 전체 할일 표현 방식

- 프로젝트 루트는 사용자 입장에서 중복 정보라 목록에서 생략
- 첫 번째 보이는 위계의 큰 묶음 사이 간격을 크게 둬서 그룹을 구분
- 각 노드는 마인드맵 깊이 색상을 그대로 사용

### 3-3. 테마 정책

- 트레이/목록모드의 빠른 다크모드 버튼은 제거
- 테마 선택은 설정창의 `시스템 / 라이트 / 다크`만 사용
- 시스템 모드는 OS `prefers-color-scheme`를 따름

### 3-4. 광고 상태

- 광고 슬롯 및 앱 안내 영역은 현재 화면에서 숨김 처리
- AdSense 관련 리소스는 완전히 제거하지 않고 보류 상태로 유지
