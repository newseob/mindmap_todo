마인드맵 할일 앱 안내

이 웹앱은 할 일을 마인드맵 형태로 정리하고, 서로 연결된 작업 구조를 한 화면에서 관리할 수 있도록 만든 생산성 도구입니다. 프로젝트별로 작업을 나누고, 노드 단위로 아이디어와 실행 항목을 시각적으로 확장할 수 있습니다.

사용자는 새 프로젝트를 만들고, 노드를 추가하거나 연결을 해제해 독립 작업으로 바꾸고, 중요한 항목을 오늘할일로 표시해 우선순위를 정리할 수 있습니다. 메모 영역에는 현재 프로젝트와 관련된 참고사항을 남길 수 있습니다.

주요 기능

  • 프로젝트별 마인드맵 관리
  • 부모-자식 노드 구조와 독립 노드 배치
  • 오늘할일 표시와 우측 요약 목록 확인
  • 메모 작성, 노드 이동, 연결 변경, 삭제 기능 제공

이용 방법

  • 오른쪽 프로젝트 영역에서 현재 프로젝트를 선택하거나 새 프로젝트를 추가합니다.
  • 노드를 선택한 뒤 하단 기능 버튼이나 우클릭 메뉴로 새 노드를 추가합니다.
  • 빈 공간 우클릭으로 연결선 없는 독립 노드를 만들고 자유롭게 배치할 수 있습니다.

운영 안내

이 페이지는 사용자가 자신의 작업과 아이디어를 구조적으로 정리할 수 있도록 제공되는 웹 기반 도구 화면입니다. 광고가 표시되는 경우에도 앱의 주요 목적은 작업 관리와 정보 정리에 있습니다.

개인정보처리방침

이 웹앱은 브라우저 로컬 저장소를 이용해 프로젝트 데이터와 메모를 기기 안에 저장합니다. 사용자가 직접 입력한 데이터는 별도 회원 데이터베이스로 전송하지 않으며, 광고 제공 과정에서 제3자 서비스가 쿠키나 기기 정보를 처리할 수 있습니다.

문의

운영 문의나 광고 관련 요청이 필요하면 서비스 운영자에게 별도 연락 채널을 제공할 예정입니다. 현재 화면은 앱 기능 설명과 사용 흐름을 안내하기 위한 기본 소개 영역을 포함합니다.

이 안내 페이지는 서비스 소개와 정책 정보를 한눈에 확인할 수 있도록 별도 창으로 제공됩니다.
